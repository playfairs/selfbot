### How to use the selfbot

```Bash
git clone https://github.com/playfairs/selfbot.git

```
```Bash
cd selfbot
```

##### For Nushell
```Bash
bash -c "source selfbot/bin/activate && exec bash"
```
```Bash
pip install -r requirements.txt
```
```Bash
cd src
```
```Bash
python main.py
```
Or
```Bash
python3 main.py
```

##### For Bash and Zsh
```Bash
source selfbot/bin/activate
```
```Bash
pip install -r requirements.txt
```
```Bash
cd src
```
```Bash
python main.py
```
Or
```Bash
python3 main.py
```
